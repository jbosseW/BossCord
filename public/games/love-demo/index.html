How LOVE2D games run in the browser

LOVE2D games are written in Lua and use the LOVE framework for 2D rendering, audio, and input. To run in a browser, they are compiled to JavaScript/WebAssembly using love.js, an Emscripten-based port of LOVE.

The build process takes .lua source files, packages them into a .love archive, and compiles the LOVE engine + game into a .js + .wasm bundle that runs in an HTML5 canvas.

Once compiled, the game communicates with BossCord via the same bridge.js postMessage API used by Phaser and Babylon.js games. The Lua side calls JavaScript bridge functions through love.js's JS interop.
